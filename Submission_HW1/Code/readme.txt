To Run this file:

Step1: Download the "Submission" folder and extract its files in same folder as DataSet.
Step2: Open run_me.py file, and set Path to the folder where "Submission" and "Data" are present.
Step3: Run run_me.py file, All results will get stored in designated folder inside Submission Folder.


Note: If you are running it through Anaconda CMD, please close the Figure to proceed further.
Desired IDE: Canopy


Important Notes: K-Fold CrossValidation has been done on each dataset and output files have been saved in Predictions folder.
